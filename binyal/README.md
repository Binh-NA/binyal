# Binyal - UI Kit `binyal`

Create simple template component support for your project.
[Live demo](https://binyal.xyz/)

## Components

- Button
- GroupButton
- CheckBox
- Radio
- Flex
- Input
- Select
- Toggle
- Loading
- Rating
- Space
- Tooltip

## Futures

- Table

## Example Usage

```
import { Button } from "binyal";
```
